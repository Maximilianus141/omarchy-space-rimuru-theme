# Omarchy Space Rimuru Theme

A custom Omarchy/Hyprland theme based on Rimuru Tempest (That Time I Got Reincarnated as a Slime).

## Palette
- **Void Black:** #0B0C16
- **Tempest Cyan:** #4DBEDF
- **Demon Lord Gold:** #FFD700
- **Cloud White:** #EEF1F5

## Installation
Clone this into your themes directory:
`git clone https://github.com/maximilianus141/omarchy-space-rimuru-theme.git ~/.config/omarchy/themes/space-rimuru`
